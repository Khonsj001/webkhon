ຊ່ຽວຊານໃນພາສາຕ່າງໆເຊັ່ນ: HTML, CSS, JavaScript, Python, PHP, ແລະ ອື່ນໆອີກຫຼາຍພາສາ.

ປະສົບການການເຮັດວຽກ

ເຮັດວຽກກັບບໍລິສັດຫຼາຍແຫ່ງ ແລະ ພັດທະນາໂຄງການຕ່າງໆມາຫຼາຍໂຄງການ.

ຊີວະປະຫວັດໂດຍຫຍໍ້

ສະບາຍດີ, ຂ້ອຍຊື່ ທ້າວຄອນເເສງຈັນ. ຂ້ອຍຊ່ຽວຊ ານດ້ານການຂຽນໂປຣແກຣມແລະການພັດທະນາເວັບໄຊທີ່ມີຄວາມຫລົງໄຫລໃນການຮຽນຮູ້ກ່ຽວກັບໂລກຂອງການຂຽນໂຄດ. ຂ້ອຍມີປະສົບການໃນການພັດທະນາຊອບແວຫຼາຍປີແລະໄດ້ໃຊ້ຄວາມຮູ້ຂອງຂ້ອຍເພື່ອຊ່ວຍໃຫ້ຕົວເອງກາຍເປັນນັກພັດທະນາທີ່ດີຂຶ້ນ.

ຂ້ອຍມີຄວາມຊ່ຽວຊານໃນພາສາຕ່າງໆເຊັ່ນ: HTML, CSS, JavaScript, SQL, Python, PHP, C++, C#, C ແລະ Bootstrap. ຂ້ອຍເຊື່ອວ່າການຮຽນຮູ້ການຂຽນໂຄດຄວນເປັນເລື່ອງທີ່ສະໜຸກແລະເຂົ້າໃຈງ່າຍ, ນັ້ນຄືເຫດຜົນທີ່ຂ້ອຍພັດທະນາວິທີການຮຽນທີ່ເນັ້ນການເຮັດຕົວຈິງແລະໂຄງການຕ່າງໆ.
